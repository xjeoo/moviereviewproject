# Movie Review Project

Un site simplu pentru review-uri de filme, creat ca proiect de invatare in web development.

## Functionalitati
- Vizualizare liste de filme
- Adaugare, citire, editare si stergere de review-uri

## Tehnologii folosite
- React
- Express
- MySQL
