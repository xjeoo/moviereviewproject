# Movie Review Project

Un site simplu pentru review-uri de filme, creat ca proiect de învățare în web development.

## Funcționalități
- Vizualizare liste de filme
- Adăugare și citire review-uri

## Tehnologii folosite
- React
- Express
- MySQL
